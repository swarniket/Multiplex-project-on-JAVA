I have provided my project in rar file which is devided into two parts. If you don't want any issues related missing files 
or corrupted file then extract both the rar files at the same time. You can do this as follows-

1. Select both the rar files while pressing the ctrl key and selecting the rars.
2. After step 1 leave the ctrl key and right click on the rar files.
3. After right clicking select extract files option.
